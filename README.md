# Web Traffic Analytics using Google Analytics 4

## Objective
To analyze website traffic and understand user behavior using Google Analytics 4.

## Tools Used
- Google Analytics 4
- Google Merchandise Store Demo Account
- GitHub

## Metrics Analyzed
- Page Views
- Traffic Sources
- Drop-Off Points
- User Journey

## Dashboard Overview
- GA4 dashboard provided an overview of website traffic and engagement.
- User activity and engagement were observed through reports.

## Page Views Analysis
- Homepage received the highest number of views.
- Users mainly interacted with important pages.

## Traffic Source Analysis
- Direct traffic was the major traffic source.
- Organic Search also contributed significant visitors.

## Drop-Off Analysis
- Certain landing pages showed lower engagement.
- Possible user exits were identified.

## User Journey Analysis
- Path Exploration showed how users moved through the website.
- Navigation patterns and exits were observed.

## Key Findings
- Homepage generated highest traffic.
- Direct traffic dominated.
- Organic Search performed well.
- Some pages had lower engagement.
- User behavior patterns were identified.

## Recommendations
- Improve low-engagement pages.
- Optimize landing page content.
- Improve navigation flow.
- Monitor website performance regularly.

## Conclusion
Google Analytics 4 helped analyze website traffic and user behavior. The findings provided useful insights to improve engagement and website performance.
